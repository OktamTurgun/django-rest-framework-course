# Examples: Filtering & Searching

Ushbu papkada **Filtering & Searching** mavzusi bo'yicha amaliy misollar keltirilgan.

---

## Fayllar

### 1. `01-basic-filtering.py`
- Manual filtering with `get_queryset()`
- URL parameters bilan ishlash
- Simple filter logic

### 2. `02-search-filter.py`
- DRF SearchFilter
- Search operators (^, =, @, $)
- Multiple field search

### 3. `03-ordering.py`
- DRF OrderingFilter
- Single va multiple ordering
- Default ordering

### 4. `04-django-filter.py`
- DjangoFilterBackend
- FilterSet class
- Lookup expressions
- Range filters
- Date filters
